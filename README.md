Install Rails
======

Install Rails is a script to set up a Linux system for Rails development.

Requirements
------------

### Linux

Currently supports Ubuntu and Linux Mint

Install
-------

### Linux

Run the script:

    bash <(wget -qO- https://raw.github.com/techAPJ/install-rails/master/linux)

What it sets up
---------------

* [Git][git_link]
* [rbenv][rbenv_link]
* [ruby-build][ruby_build_link]
* [Ruby][ruby_link] (stable)
* [Rails][rails_link]
* [PostgreSQL][pg_link]
* [SQLite][sqlite_link]
* [Redis][redis_link]
* [Bundler][bundler_link]

It should take less than 15 minutes to install (depends on your machine).
